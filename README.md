# Cypress QA Automation Framework

## Project Overview

This project demonstrates end-to-end test automation using Cypress and JavaScript.

The framework covers:

- Login functionality
- Sidebar navigation
- Transaction validation
- Favourite fruit management
- Settings persistence
- Logout functionality

## Tech Stack

- Cypress
- JavaScript
- Git
- GitHub

## Test Coverage

### Login
- Valid login
- Invalid login
- Empty credentials

### Transactions
- Table validation
- Pagination validation
- Data verification

### Favourite Fruit
- Move fruit to favourites
- Verify container updates

### Settings
- Logo preference persistence
- Refresh validation
- Cross-page validation

### Logout
- Session cleanup
- Redirect validation

## Installation

npm install

## Run Tests

npx cypress open

## Run Headless

npx cypress run
